# 📱 GLI Platform - Panduan Pengguna

## 🚀 Quick Start (5 Menit)

### 1️⃣ **Register & Login**
```
1. Klik "Daftar" di halaman login
2. Masukkan email & password
3. Atau login dengan Google
4. Selesai! ✅
```

### 2️⃣ **Explore Dashboard**
- **Poin:** Lihat total poin Anda
- **Level:** Eco-Newbie → Eco-Warrior → Eco-Champion
- **Aksi Pending:** Status aksi yang menunggu verifikasi
- **Medal:** Badge yang sudah dikumpulkan

---

## 👤 **User Guide**

### **1. Submit Aksi (Buat Aksi Baru)**

**Langkah:**
1. Dashboard → Buat Aksi
2. Pilih tipe aksi (Hemat Energi, Hemat Air, Daur Ulang, Tanam Pohon)
3. Upload foto bukti
4. Klik Submit
5. Tunggu admin approve ✅

**Tips:** Foto harus jelas dan menunjukkan aksi nyata

---

### **2. Lihat Riwayat Aksi**

**Langkah:**
1. Sidebar → Riwayat
2. Lihat status aksi (Pending/Approved/Rejected)
3. Klik aksi untuk detail
4. Lihat feedback dari admin

---

### **3. Cek Leaderboard (Peringkat)**

**Langkah:**
1. Sidebar → Peringkat
2. Lihat ranking berdasarkan poin bulanan
3. Hanya user dengan aksi/poin yang muncul
4. Ranking update otomatis setiap bulan

---

### **4. Kumpulkan Medal**

**Cara Dapatkan Medal:**
- Submit & dapatkan approve ✅ → Dapat Medal
- Setiap tipe aksi punya medal berbeda:
  - Hemat Energi → ⚡ PAHLAWAN ENERGI
  - Hemat Air → 💧 HEMAT AIR
  - Daur Ulang → ♻️ DAUR ULANG
  - Tanam Pohon → 🌲 PENANAM POHON

**Lihat Medal:**
1. Profil → Koleksi Medal
2. Lihat semua medal yang sudah dikumpulkan

---

### **5. Ikuti Event**

**Langkah:**
1. Sidebar → Event
2. Lihat daftar event yang tersedia
3. Klik event untuk detail
4. Klik Daftar untuk ikut event
5. Tunggu event dimulai

**Saat Event Berlangsung:**
1. Hadir & ambil foto bukti kehadiran
2. Upload bukti di event
3. Admin verify ✅
4. Dapat medal otomatis

---

### **6. Chat dengan Mentor**

**Langkah:**
1. Sidebar → Mentoring
2. Lihat daftar mentor yang online 🟢
3. Klik mentor untuk mulai chat
4. Tanya-jawab real-time
5. Mentor berubah offline jika idle >5 menit

---

### **7. Edit Profil**

**Edit Nama:**
1. Profil → Klik edit icon di sebelah nama
2. Ketik nama baru
3. Klik Simpan ✅

**Upload Foto Profil:**
1. Profil → Hover di avatar
2. Klik upload icon
3. Pilih foto dari device
4. Foto tersimpan otomatis ✅

---

## 🛡️ **Admin Guide**

### **1. Dashboard**

**Lihat:**
- 📊 Traffic chart (grafik aktivitas 7 hari)
- 👥 Top 5 performers (user terbaik)
- ⏳ Recent actions (aksi pending terbaru)
- 📈 Action approval rate

**Filter:**
- Pilih tanggal/bulan/tahun untuk melihat data spesifik

---

### **2. Moderasi (Verifikasi Aksi)**

**Langkah:**
1. Sidebar → Moderasi
2. Lihat list aksi pending
3. Klik aksi untuk lihat foto bukti
4. Pilih: **Setujui** atau **Tolak**

**Jika Setujui:**
1. Masukkan poin (default: 50)
2. Klik "Setujui"
3. Medal otomatis diberikan ✅

**Jika Tolak:**
1. Masukkan alasan penolakan
2. Klik "Tolak"
3. User dapat feedback ℹ️

---

### **3. User Management (Monitoring Pengguna)**

**Langkah:**
1. Sidebar → Monitoring
2. Lihat list semua user
3. Lihat status (Online/Offline)
4. Klik user untuk melihat detail

**User Detail:**
- Total poin & monthly poin
- Level & medal
- Riwayat aksi
- Ranking

---

### **4. Manage Event**

**Buat Event Baru:**
1. Sidebar → Event
2. Klik "Buat Event"
3. Isi detail event:
   - Judul, deskripsi
   - Lokasi, WhatsApp link
   - Tanggal registrasi & event
   - Medal untuk event
   - Thumbnail (foto/teks & warna)
4. Klik "Buat" ✅

**Event Berjalan:**
1. Lihat list registrasi
2. Verifikasi bukti kehadiran user
3. Approve → Medal diberikan ✅

---

### **5. Analytics (Analitik)**

**Lihat:**
- 📊 Traffic bulanan/tahunan
- 👥 User statistics
- 🎯 Action completion rate
- 🏆 Top performers

**Export Data:**
- Download laporan dalam format CSV/PDF

## ⚠️ **Troubleshooting**

### **Aksi Tidak Approved**
- Foto harus jelas & menunjukkan aksi nyata
- Cek feedback dari admin di riwayat
- Resubmit dengan foto lebih baik

### **Medal Tidak Muncul**
- Medal muncul setelah admin approve
- Refresh halaman untuk update
- Medal akan ada di profil → koleksi medal

### **Login Gagal**
- Pastikan email & password benar
- Coba hapus cache browser
- Atau login dengan Google

### **Chat Mentor Tidak Bisa**
- Mentor harus online 🟢
- Jika idle >5 menit, akan auto-offline
- Coba chat mentor lain yang online

### **Event Tidak Daftar**
- Pastikan periode registrasi belum tutup
- Cek tanggal registrasi di detail event
- Hubungi admin jika ada masalah

## 📋 **Poin & Reward**

### **Sistem Poin**
- Setiap aksi approve = mendapat poin (biasanya 50)
- Poin bulanan di-reset setiap 1 Januari
- Total poin tidak pernah di-reset

### **Level System**
- **Eco-Newbie** (0-50 poin)
- **Eco-Warrior** (51-150 poin)
- **Eco-Champion** (150+ poin)

### **Leaderboard**
- Ranking berdasarkan poin bulanan
- Update otomatis
- Hanya user dengan poin > 0 yang muncul
- Reset setiap bulan

---

## 🔒 **Keamanan**

### **Password**
- Jangan bagikan password ke orang lain
- Gunakan password yang kuat
- Logout setelah selesai

### **Data Pribadi**
- Tidak akan dibagikan ke pihak ketiga
- Hanya untuk verifikasi aksi
- Foto bisa di-delete kapan saja

### **Upload Foto**
- Hanya foto yang relevan dengan aksi
- Ukuran max: 5MB
- Format: JPG, PNG, JPEG
